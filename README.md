# Dark Credit Card

Uma interface elegante e minimalista de um cartão de crédito premium, desenvolvida com HTML5 e CSS3. O projeto foca-se na estilização avançada de superfícies escuras, no uso de padrões de textura de fundo e no alinhamento preciso de elementos de interface financeira.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica dos dados do cartão (número, titular, validade).
* **CSS3:** Estilização baseada num esquema de cores escuro (*Dark Mode*).
* **CSS Flexbox / Grid:** Organização e distribuição dos elementos internos do cartão.
* **Texturas de Fundo:** Aplicação de padrões ondulados sutis para um acabamento sofisticado.

## Funcionalidades e Técnicas Aplicadas

* **Fundo Texturizado:** Uso de imagens de fundo ou gradientes para criar um efeito de ondas abstratas e dinâmicas na superfície do cartão.
* **Estilização de Elementos Reais:** Renderização visual de componentes como o chip dourado e o logótipo da bandeira do cartão.
* **Tipografia e Contraste:** Escolha de fontes e cores que garantem uma excelente legibilidade sobre um fundo puramente escuro.
* **Bordas Arredondadas e Sombras:** Aplicação de `border-radius` e `box-shadow` para criar um aspeto realista e flutuante sobre a página.
